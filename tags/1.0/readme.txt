== Description ==

New Muslims aspires to be a unique interactive and informative online resource about Islam for new Muslims as well as potential converts. All what you have to do is to install the Plugin, then the content will be stored automatically in your database. Afterwards, the plugin will work automatically using Cron Job to get the recent updates daily or according to the time you set.

Features:

* Getting the content (articles, videos, audios) of [New-muslims.info](http://www.new-muslims.info/).
* Using Json API
* Using Cron Job


PS: All the websites owned by EDC [(E-Da`wah Committee)](http://www.edc.org.kw), For Support click to [Islam.com.kw](http://www.islam.com.kw/).


== Installation ==

* Upload the Plugin to your blog, Activate it.
* Then the content will be stored automatically in your database.
* The plugin will work automatically using Cron Job to get the recent updates daily or according to the time you set.
